Google Sheets Integration Setup

The Gurukul High · Admissions System

How It Works

When a parent submits the Admission Form on your website, the server saves the data to data/admissions.json locally. It also automatically sends a copy to your Google Sheet — so you can view all applications in a spreadsheet without logging into the admin panel.

This uses Google Apps Script (a free Google tool). No Google Cloud billing, no OAuth2 setup — just a small script you add to your Google Sheet once.

1  Create Your Google Sheet

  1. Go to sheets.google.com and create a new spreadsheet.
  2. Name it "Gurukul Admissions 2026–27" (or anything you like).
  3. Rename the first tab (sheet) to Admissions.
💡 The script will automatically add a header row and all columns on the first submission.

2  Add the Apps Script

  1. In your Google Sheet, click Extensions → Apps Script.
  2. Delete any existing code in the editor.
  3. Paste the following script and click Save (Ctrl+S):
// Gurukul Admissions – Google Apps Script Web App
// Paste this in Extensions → Apps Script, then deploy as web app

const SHEET_NAME = 'Admissions';
const HEADERS = [
  'App ID', 'Date & Time', 'Status',
  'First Name', 'Last Name', 'Date of Birth',
  'Gender', 'Blood Group', 'Grade Applying',
  'Previous School', 'Last %',
  "Father's Name", "Father's Mobile", "Father's Email", "Father's Occupation",
  "Mother's Name", "Mother's Mobile",
  'Address', 'City', 'PIN',
  'Heard About Us', 'Reason'
];

function doPost(e) {
  try {
    const d = JSON.parse(e.postData.contents);
    if (d.action !== 'addAdmission') return ok({ skipped: true });

    const ss    = SpreadsheetApp.getActiveSpreadsheet();
    const sheet = ss.getSheetByName(SHEET_NAME) || ss.insertSheet(SHEET_NAME);
    if (sheet.getLastRow() === 0) sheet.appendRow(HEADERS);

    const s = d.data;
    sheet.appendRow([
      s.id, s.submittedAt, s.status,
      s.firstName, s.lastName, s.dob,
      s.gender, s.bloodGroup, s.gradeApplying,
      s.prevSchool, s.lastPercentage,
      s.fatherName, s.fatherMobile, s.fatherEmail, s.fatherOccupation,
      s.motherName, s.motherMobile,
      s.address, s.city, s.pin,
      s.hearAbout, s.reasonAdmission
    ]);
    return ok({ success: true, id: s.id });
  } catch(err) {
    return ok({ error: err.message });
  }
}

function ok(data) {
  return ContentService
    .createTextOutput(JSON.stringify(data))
    .setMimeType(ContentService.MimeType.JSON);
}

3  Deploy as Web App

  1. Click Deploy → New deployment.
  2. Click the gear icon ⚙ next to Select type and choose Web app.
  3. Set Execute as → Me.
  4. Set Who has access → Anyone.
  5. Click Deploy. Google will ask for permissions — click Allow.
  6. Copy the Web app URL (looks like https://script.google.com/macros/s/ABC123.../exec).
⚠️ Keep this URL private — anyone with it can add rows to your sheet. Don't share it publicly.

4  Add the URL to Your Server

Create a file called .env inside the server/ folder (next to server.js). Add this line:

# Paste your Apps Script Web App URL here:
APPS_SCRIPT_URL=https://script.google.com/macros/s/YOUR_SCRIPT_ID/exec

Then restart the server:

node server/server.js

The server reads .env automatically. From now on, every new admission form submission will appear in your Google Sheet within seconds.

✅ The server already supports this — no code changes needed. Just create .env and restart.

5  Test It

  1. Open the Admission Form and submit a test application.
  2. Check your Google Sheet — a new row should appear within 2–3 seconds.
  3. If it doesn't appear, check the server console for any warning messages about the Apps Script URL.

Optional: Load Portal Data from Google Sheets

You can also keep student data (attendance, marks, fees) in Google Sheets and sync it to the portal. Set these in your .env file:

# Student portal data (read from public Google Sheet)
GOOGLE_SHEET_ID=your_sheet_id_here
GOOGLE_API_KEY=your_google_api_key_here

Then call POST /api/admin/sync-sheets to pull data from the sheet into the portal's JSON files. The sheet must have tabs named Students, Attendance, Marks, Fees with the correct column headers.

💡 To get a Google API key: go to Google Cloud Console → Enable Google Sheets API → Create API key → Restrict it to Sheets API only.
